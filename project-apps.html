What

Accelerator Pedal Position Sensor

Built a variable signal conditioning circuit that converts variable resistor signals from the accelerator pedal into clean, amplified voltage outputs for the MoTeC ECU of FSAE car. Features dual independent signal input to meet Formula SAE safety requirements where the 2 signals are compared and if the deviation exceeds 10%, the accelerator is cut off.

  • Signal Processing

    Variable Resistor → Buffer → Variable Offset → Variable Gain

  • Dual Redundancy

    Two independent sensor channels with 10% deviation monitoring

  • Tunable Parameters

    Potentiometer knobs for offset and gain adjustment

  • Accelerator Cut-off

    Automatic signal cut when deviation exceeds 10% threshold
