Autonomous Mobile Robot

Developed a sensor-fusion control system integrating multiple sensors with PI controllers onto a mobile robot for autonomous navigation and area coverage.

  • C++/C
  • Control Systems
  • Sensor Fusion
  • SLAM

01

What

Self-Driving Robot

Built an autonomous robot as a university project that can navigate around obstacles, map unknown areas, extinguish artifical fires and complete tasks without human control. It uses multiple sensors and omnidirectional wheels.

  • Autonomous Navigation

    Complex environment navigation with obstacle avoidance

  • EKF SLAM Implementation

    Simultaneous localization and mapping in real-time

  • Omnidirectional Movement

    Mecanum wheels for precise movement control

  • Sensor Fusion

    IR, ultrasonic,, gyroscope, phototransistor data integration
